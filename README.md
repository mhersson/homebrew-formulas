# Homebrew Formulas

Homebrew tap for [my](https://github.com/mhersson) projects.

## Installation

```bash
brew tap mhersson/formulas
brew install mpls
brew install vectorsigma
```

## Available Formulas

- mpls - Markdown Preview Language Server with live browser preview
- vectorsigma - VectorSigma is a tool for generating finite state machines (FSM)
  in Go from UML diagrams.
